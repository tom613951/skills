开放式 Agent 技能生态系统

为 Claude Code, Cursor, Windsurf, Roo Code 等主流 AI 终端 Agent 提供统一的、开箱即用的技能安装与管理工具。

常用命令 查看源码
# 安装一个技能到您的 Agent 目录中
npx skills add vercel-labs/agent-skills

# 交互式查找并添加技能
npx skills find

核心特性

⚡ 零配置安装

无需繁琐的参数配置，Skills CLI 会自动检测您本地安装的 AI 编辑器与终端 Agent，并以符号链接（symlink）或文件复制的方式将技能写入目标目录。

🔍 交互式技能搜索

输入关键词，在包含数百个优质社区技能的 skills.sh 技能库中进行过滤、多选并一键同步到本地。

📦 多客户端同步

同一个项目中的技能，可以一次性同步到 Cursor、Claude Code 等多个不同的客户端，实现团队开发环境的统一。

命令行参考

skills add <pkg> 从 Git 仓库、URL 或本地路径添加指定的技能至 Agent 目录
skills remove [skills...] 从 Agent 中卸载已安装的技能 (支持多选及 --all 清除)
skills list / ls 列出当前项目或全局已安装的所有技能，并显示所关联的 Agent
skills find [query] 在远程注册表中进行交互式检索，并选择性安装技能
skills update [skills...] 检查锁文件，一键更新所有已被修改的技能到最新版本
skills init [name] 初始化一个新技能目录，生成自定义的 SKILL.md 模版
